# Customer Churn Prediction

##  Project Overview
This project focuses on predicting customer churn for a telecom company using machine learning classification techniques.

##  Dataset
**Telecom Customer Churn Dataset**
Features include:
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Payment Method

##  Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

##  Key Analysis
- Categorical variable encoding
- Churn vs tenure visualization
- Statistical comparison between churned and retained customers

##  Model Used
- Logistic Regression

##  Results
- High churn observed among customers with lower tenure
- Monthly charges play a significant role in churn prediction

##  How to Run
1. Clone the repository
2. Install dependencies
3. Open the notebook
4. Run all cells
